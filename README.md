# pug-boilerplate
Simple Webpack5 + Pug boilerplate

<hr/>

### 1. Clone repository
<code>git clone https://github.com/AvetisDN/pug-boilerplate.git
</code>
### 2. Install dependencies
<code>npm i</code>
### 3. Scripts
#### 3.1. Dev Server
<code>npm run serve</code>
#### 3.2. Build a distribution
<code>npm run buil</code>
